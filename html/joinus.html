Installer GIT pour Windows ---------> https://git-scm.com/downloads

Se créer un compte sur GitHub ------> https://github.com/

Rendez-vous sur votre boîte mail pour cliquer sur le lien d'invitation

Créer un répertoire sur le bureau et nommez-le github

Ouvrez une invite de commande en tapant CMD dans le menu démarrer de windows

git --version_________/*Si cette commande retourne le numéro de version, Git est bien installé

git clone URL_________/*Remplacer URL par le lien du repository github

git add --all_________/*Permet de charger les fichiers après modification

git commit -m "Initial commit"_________/*Permet d'identifier les modifications

git push -u origin master_________/*Propulse les fichiers modifiés sur le repository

Aller plus loin grâce au git book   Notre dépôt

Les principales commandes utiles sous windows

cd_____________________________________/*se positionner dans un répertoire

dir_____________________________________/*voir le contenu d'un répertoire

tree____________________________________/*voir l'arborescence à partir d'un répertoire

cd .._____________________________________/*revenir en arrière

cd /._____________________________________/*revenir à la racine

copy_____________________________________/*copier un répertoire ou un fichier

copy NOM DU REPERTOIRE/*.*________________/*copier l'intégralité d'un répertoire

robocopy__________________________________/*copier un répertoire et ses fichiers

mkdir_____________________________________/*créer un répertoire

start NOM_DU_PROGRAMME___________________/*permet de démarrer un programme

cls_________________________________________/*clean le terminal

del__________________________________________/*permet de supprimer un fichier

rd ou rmdir____________________________________/*permet de supprimer un dossier

takeown_______________________________________/*permet de devenir propriétaire des fichiers

icalcs_________________________________________/*permet de redéfinir les permissions de fichiers, vous devez être propriétaire des fichiers pour cela.
